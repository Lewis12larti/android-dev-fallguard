# FallGuard Cameroun
**Elderly Fall Detection & Emergency Alert System**

> Android Development —  Project | 2025–2026

---

##  Project Description
FallGuard Cameroun is an Android application that automatically 
detects falls in elderly people using smartphone sensor fusion 
(accelerometer + gyroscope + GPS) and immediately alerts 
family members — via internet or SMS fallback.

## Repository Structure
- `assignments/` → Course assignments
- `project/` → FallGuard Cameroun Android App (Kotlin)

## 🔧 Sensors Used
Accelerometer, Gyroscope, GPS, Microphone, Camera, GSM (SMS)
